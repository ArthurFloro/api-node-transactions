# RF 

- [ ]O usuário deve poder criar uma nova transação; 
- [ ]O usuário deve poder ter um resumo da sua conta;
- [ ]O usuário deve poder listar todas as suas transações que já ocorreram;
- [ ]O usuário deve poder visualizar uma transação única

# RN 

- [ ]A transação pode ser do tipo crédito que somará ao valor total, ou débito que subtrairá ao valor total; 
- [ ]Deve ser possível identificarmos o usuário entre as requisições
- [ ]O usuário so pode visualizar transações o qual ele criou
